General Controls

XML Banner

Selecting to remove the XML banner will remove the Splunk banner from SimpleXML dashboards.

JSON Banner

Selecting to remove the XML banner will remove the Splunk banner from the new JSON based dashboards, currently seen in the Dashboards Beta app.

Presentation Controls

Seconds

Set the number of seconds before automatically switching to the next tab. This is extremely useful when you want to rotate between multiple dashboards on a single pane of glass.

Toggle

Press the toggle button to enable or disable the presentation mode (auto-rotate)
